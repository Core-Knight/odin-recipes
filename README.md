# Odin Recipes
A recipe website project from The Odin Project curriculum.

# Description
This project demonstrates fundamental HTML and CSS (in progress) coding skills through building a basic recipe website. It's part of The Odin Project's Foundations course.

# Skills Demonstrated in
 HTML: Semantic markup, links, images, lists.
 CSS: Styling, layouts, colors (to be added).

# Project Status so far
Currently completed: 3 out of 3 recipes added.

# About
This is a foundational project for learning web development through hands-on practice.
